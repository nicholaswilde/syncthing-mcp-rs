# Specification: GUI & Web Security Management

## Objective
Provide tools to manage SyncThing's GUI and web-related security settings.

## Requirements
- Implement `GET` and `PUT/PATCH` for `/rest/config/gui`.
- Support managing themes, UI language, and security settings (HTTPS, Auth).
- Create tools to audit GUI security (e.g., "Is HTTPS enabled?").
- Comprehensive unit and integration tests.
